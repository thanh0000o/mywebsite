Create an animated flower that grows as you scroll, with clickable portfolio pieces:
Flower Design:

Flower grows from center of canvas (where logo was initially)
12-15 curved stems radiating outward organically
Each stem has a small square (14x14px) at the endpoint
Green color: #00AA00 or #00FF00
Match the organic, flowing design from the reference image

Growth Animation:

Flower starts growing after logo reaches top-left (~300px scroll)
Stems grow progressively from center outward
Growth is pixelated/stepped (update every 4-8px)
Each stem grows at slightly different speeds for organic feel
Squares appear when stem is 100% grown
Full flower visible by 1000-1200px of scrolling

Scroll-Based Growth:
javascriptcanvas.addEventListener('scroll', function() {
  const scrollY = canvas.scrollTop;
  
  // Start flower growth after logo animation (300px)
  if (scrollY > 300) {
    const flowerProgress = Math.min((scrollY - 300) / 900, 1); // 300-1200px range
    
    // Pixelate progress (update every 2% for stepped effect)
    const pixelatedProgress = Math.floor(flowerProgress * 50) / 50;
    
    // Animate each stem using stroke-dasharray/stroke-dashoffset
    stems.forEach((stem, index) => {
      const length = stem.getTotalLength();
      const delay = index * 0.05; // stagger each stem
      const stemProgress = Math.max(0, Math.min(1, (pixelatedProgress - delay) / 0.8));
      
      stem.style.strokeDasharray = length;
      stem.style.strokeDashoffset = length * (1 - stemProgress);
    });
    
    // Show squares when stems are grown
    squares.forEach((square, index) => {
      if (pixelatedProgress > 0.7 + (index * 0.02)) {
        square.style.opacity = 1;
      }
    });
  }
});
