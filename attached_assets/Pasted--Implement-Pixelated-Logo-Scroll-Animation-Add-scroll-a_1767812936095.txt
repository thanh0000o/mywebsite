 Implement Pixelated Logo Scroll Animation
Add scroll animation that moves logo to top-left corner:
Animation Behavior:

Logo starts centered (already positioned from Prompt 2)
When user scrolls in .canvas-content, logo moves to top-left
Animation completes in first 300px of scrolling
Movement is pixelated (4px steps) for retro feel
Final position: 20px from top, 20px from left
Logo stays fixed after reaching top-left

JavaScript Implementation:
javascript// Get elements
const canvasContent = document.querySelector('.canvas-content');
const logo = document.querySelector('.logo-container');

// Get initial center position
const canvasWidth = canvasContent.offsetWidth;
const startX = canvasWidth / 2; // Center
const startY = 300; // Initial top position

// Target position (top-left)
const endX = 20;
const endY = 20;

// Animation settings
const maxScroll = 300; // Animation duration in pixels

// Scroll event listener
canvasContent.addEventListener('scroll', function() {
  const scrollY = canvasContent.scrollTop;
  
  // Calculate progress (0 to 1)
  const progress = Math.min(scrollY / maxScroll, 1);
  
  // Calculate new position
  let newX = startX + (endX - startX) * progress;
  let newY = startY + (endY - startY) * progress;
  
  // PIXELATE: Round to nearest 4px for stepped movement
  newX = Math.round(newX / 4) * 4;
  newY = Math.round(newY / 4) * 4;
  
  // Apply position
  if (progress < 1) {
    // During animation: use left positioning
    logo.style.left = newX + 'px';
    logo.style.top = newY + 'px';
    logo.style.transform = 'none';
  } else {
    // After animation: fixed at top-left
    logo.style.left = endX + 'px';
    logo.style.top = endY + 'px';
    logo.style.transform = 'none';
  }
});
